Downloads

Public files can be shared from this site.

How it works

Put files into source/downloads/files/ in the Hexo source repo.

After the site is generated, each file will be available at:

https://www.taki-tani.com/downloads/files/your-file-name.ext

Notes

  • Everything here is public.
  • Small files are a good fit for the site.
  • Large or frequently updated files are better hosted in GitHub Releases.

Folder

The upload folder is prepared here:

source/downloads/files/

When you want to publish a file, send it to me and I can place it there and update this page with a direct link.
